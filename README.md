# Laboratory Sales & Inventory Analysis using SQL

## Project Overview
This SQL project analyzes laboratory sales and inventory data to generate business insights related to revenue, product performance, and stock management.

The project simulates healthcare/laboratory workflows and demonstrates SQL data analysis skills.

## Analysis Performed
Total revenue calculation
Top-selling products
Department wise revenue analysis
Low inventory tracking
Category-wise sales performance

## SQL Concepts Used
CREATE DATABASE
CREATE TABLE
INSERT INTO
SELECT
WHERE
GROUP BY
ORDER BY
Aggregate functions (SUM)
  
##  Key Insights
Molecular category generated high revenue
PCR Kits showed strong sales performance
Certain equipment products had low inventory levels
